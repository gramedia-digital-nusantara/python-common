Welcome to GDN Python Common's documentation!
=============================================

Common utility classes for python, django, and django-rest-framework
used at PT Gramedia Digital Nusantara.

Installation
------------

This package is available on PyPI as gdn-python-common.

It can be installed with

.. code-block:: bash

    pip install gdn-python-common

    # or you can also install this with django and drf
